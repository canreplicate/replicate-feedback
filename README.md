# Replicate — Feedback & Issues
 
Public issue tracker for **[Replicate](https://canreplicate.app)** — save and restore iOS Simulator app state as a single `.replica` file.
 
## Opening an issue
 
- Search before opening — someone may have already reported it
- Include your Replicate version, macOS version, and framework (Flutter / React Native)
- For bugs: steps to reproduce + what you expected vs what happened
 
## Security
 
Don't open public issues for security vulnerabilities. Email [security@canreplicate.app](mailto:security@canreplicate.app) instead.
